# VMD - Virtual MIDI Device

A simple MIDI routing virtual instrument plugin for macOS (AUv2).

## Description

VMD (Virtual MIDI Device) is a virtual instrument plugin that routes MIDI signals. It is designed to be a simple and reliable MIDI routing solution for your DAW.

## System Requirements

- macOS 10.15 or later
- DAW that supports AUv2 plugins

## Installation

Download the installer package from the [Releases](https://github.com/kurogedelic/vmd/releases) page and run it.

## Usage

1. Open your DAW
2. Add a new software instrument track
3. Select "VMD" from the AUv2 instrument plugins
4. Route MIDI as needed
